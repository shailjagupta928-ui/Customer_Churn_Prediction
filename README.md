# 🚀 Customer Churn Prediction App

A Machine Learning project that predicts whether a customer is likely to churn (leave a service) or not. This helps businesses take proactive steps to improve customer retention.

---

## 📌 Project Overview
Customer churn is a major challenge for businesses. In this project, we analyze customer data and build a predictive model to identify customers who are at risk of leaving.

The project covers the full ML lifecycle:
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building & Evaluation
- Deployment using Streamlit

---

## 🎯 Objective
To build a model that accurately predicts customer churn and provides insights that can help businesses reduce customer loss.

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  
- Streamlit  

---

## 📊 Features
- Cleaned and preprocessed dataset  
- Performed detailed EDA to find churn patterns  
- Built multiple ML classification models  
- Selected best model based on performance metrics  
- Deployed interactive web app using Streamlit
